# Mermaid Explorer

A focused Mermaid viewer for navigating large diagrams comfortably.

## Features

- Smooth wheel zoom centered on the cursor
- Drag-to-pan hand mode
- Box zoom mode for drawing a selection rectangle over the graph
- Quick `Shift` + drag box zoom from hand mode
- Minimap recentering
- Local Mermaid rendering with bundled fonts and no CDN dependency

## Run

```bash
npm install
npm run dev
```

Then open `http://127.0.0.1:4173/`.

## Build

```bash
npm run build
```
